Proyecto 1

A CUDA implementation of the Point In Polyhedron (PIP) problem. PIP is an important problem that have a number of applications in different domains such as computer graphics. In this project. I combine and re-implement two research papers trying to parallelize the procedure on GPU using CUDA.
